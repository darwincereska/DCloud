# DCloud

DCloud is a simple online cloud storage and Content Delivery Network using nodejs and express.js


## Features

- Sleek UI
- File Previews
- Link Sharing
- File Downloading
- Unlimited Storage
- User Accounts


## Run Locally

Clone the project

```bash
  git clone https://github.com/darwincereska/DCloud
```

Go to the project directory

```bash
  cd DCloud
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node index.js
```
